# My personal dotfiles

These are my dotfiles for Manjaro Linux with i3wm. 


## Use these dotfiles

To use these dotfiles on your own machine, fork this repository, then run the command below. Replace *username*  with your GitHub username.  

    mkdir dev && cd dev && git clone https://github.com/username/dotfiles.git && sh setup  

The setup configures Git user credentials and gives the option to create an SSH key for GitHub. The remote origin connection for the repository is also changed to SSH.
