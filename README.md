# My dotfiles #

My dotfiles for Arch Linux. Run setup script after installation. 

To push to GitHub, create a git ssh key.

Adjust monitors with arandr.

## Keybindings

## System Keybindings
* Kill focused window - **$mod+q**
* Reload configuration file - **$mod+Shift+c**
* Restart i3 - **$mod+r**
* Backlight up - **?**
* Backlight down - **?**
* Focus left - **$mod+j** or **$mod+Left**
* Focus down - **$mod+k** or **$mod+Down**
* Focus up - **$mod+i** or **$mod+Up**
* Focus right - **$mod+l** or **$mod+Right**
* Move focused widow left - **$mod+Shift+j** or **$mod+Shift+Left**
* Move focused widow down - **$mod+Shift+k** or **$mod+Shift+Down**
* Move focused window up - **$mod+Shift+i** or **$mod+Shift+Up**
* Move focused window right - **$mod+Shift+l** or **$mod+Shift+Right**
* Split horizontially - **$mod+h**
* Split vertically - **$mod+v**
* Make focused window full screen - **$mod+f**
* Change container layout to stacking - **$mod+s**
* Change container layout to tabbed - **$mod+t**
* Change container layout to toggle split - **$mod+e**
* Toggle tiling/floating on focused window - **$mod+Shift+space**
* Change between tiling/floating windows - **$mod+space**
* Change focus to parent container - **$mod+p**
* Open system control menu - **$mod+Backspace**


### App Keybindings
* Open terminal - **$mod+Return**
* Open browser - **$mod+b**
* Open VSCode - **$mod+z**
* Open Pavucontrol - **$mod+a**
* Open Discord - **$mod+d**
* Open clipboard - **$mod+c**
* Clear clipboard - **$mod+Ctrl+c**
* Take a screenshot - **$mod+Return**
* Open Arandr - **$mod+a**
